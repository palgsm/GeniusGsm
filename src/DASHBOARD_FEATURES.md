# 🚀 تحديث الداشبورد - GeniusGsm

## المميزات الجديدة

### 1. 🕐 معلومات الجلسة والـ Token في الوقت الفعلي
- **عرض معرف الجلسة**: عرض معرف الجلسة الحالي مع تشفير جزئي
- **المدة المتبقية**: عداد حي يعرض الوقت المتبقي على انتهاء الجلسة
- **وقت الانتهاء**: عرض الوقت الدقيق لانتهاء الجلسة
- **حالة التوكن**: مؤشر بصري لحالة التوكن (نشط/منتهي الصلاحية)
- **شريط التقدم**: عرض بصري للوقت المتبقي

### 2. 📊 إحصائيات الداشبورد
- **IP الحالي**: عرض عنوان الـ IP الحالي للجهاز
- **إجمالي الطلبات**: عدد الطلبات المنفذة
- **تقارير الإساءة**: عدد تقارير الإساءة المسجلة
- **الخدمات النشطة**: عدد الخدمات المتاحة حالياً

### 3. 🔧 أدوات العمل المتقدمة

#### الترتيب التلقائي
- زر **ترتيب تلقائي** يفعل ترتيب البيانات حسب الأحدث
- اختيار الترتيب من القائمة المنسدلة:
  - الأحدث أولاً
  - الأقدم أولاً
  - حسب الفئة
  - حسب المرسل

#### البحث والتصفية
- حقل بحث فوري يصفي النتائج أثناء الكتابة
- زر تصفية متقدم (قريباً)

#### تصدير البيانات
- تصدير جميع البيانات إلى ملف CSV
- حفظ البيانات برايد واضح

### 4. 👁️ عرض الأجهزة والأنشطة
- **الأجهزة المتصلة**: عرض الأجهزة المتصلة حالياً
- **آخر الأنشطة**: سجل بآخر الأنشطة على الحساب

### 5. 🔔 تنبيهات الجلسة
- **تحذير قبل الانتهاء**: تنبيه قبل 5 دقائق من نهاية الجلسة
- **استمرار الجلسة**: خيار لاستمرار الجلسة الحالية

## واجهة API الجديدة

### `/api/session/`
الحصول على معلومات الجلسة الحالية:
```json
{
  "session_key": "abc123...",
  "remaining_seconds": 1800,
  "remaining_minutes": 30,
  "session_timeout_minutes": 30,
  "expiry_time": "2026-04-07T15:30:00",
  "status": "active"
}
```

### `/api/dashboard/`
الحصول على إحصائيات الداشبورد:
```json
{
  "total_reports": 42,
  "recent_reports": [...],
  "active_services": 3,
  "timestamp": "2026-04-07T15:00:00"
}
```

### `/api/token/`
التحقق من حالة التوكن:
```json
{
  "authenticated": true,
  "username": "admin",
  "session_key": "abc123...",
  "current_ip": "192.168.1.1",
  "timestamp": "2026-04-07T15:00:00"
}
```
